MemoryBox

MemoryBox - Server

Backend do sistema Memory Box, implementado em Node.js e Express.js. Oferece funcionalidades CRUD com autenticação via JWT, segurança de senhas com bcrypt e utiliza CORS para comunicação entre cliente e servidor, armazenando dados em JSON.
